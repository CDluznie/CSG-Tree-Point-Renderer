Detailed Description

Structure defining a CSG tree.

A CSG tree is a recursive structure. If the tree is a leaf, it is defined by a canonical form. If the tree is an internal node, it is defined by a combination operator, a left CSG tree child and a right CSG tree child. In both cases, the tree also contains a points transformation matrix, an inverse points transformation matrix, and a normals transformation matrix.
